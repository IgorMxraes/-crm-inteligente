# Instruções Permanentes

## Qualidade de Código
- Sempre que adicionar ou modificar um arquivo, revise o código e corrija qualquer erro automaticamente antes de finalizar.
- Garanta que o código novo é consistente com o estilo e padrões já existentes no projeto.

## Verificação Contínua
- Após cada mudança, verifique se o projeto ainda roda sem erros e corrija se necessário.
- Se encontrar algo crítico que pode quebrar o projeto de forma irreversível, avise antes de prosseguir.

## Idioma
- Sempre responda em português.
